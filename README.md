# darts-project

Scripts that I have used in my project attempting to predict the outcomes of PDC Darts matches.

## Notebooks
- This contains notebooks that I have created, attempting to showcase my work.

## Simulations
- This contains simulation functions that I used when attempting to predict the outcome of fixtures.

## Scraping
- Contains 2 functions.
- One was used to scrape match data from the PDC website.
- The other was used to extract starting prices from Betfair's historical data package.

## Processing
- Contains an example of how I created a dataframe from the scraped json data.
